```
Collection/Documentation of Port Numbers and their usages
```

## Types
> 3 types of port numbers varying in ranges
+ Well-known Ports : 0 - 1023
+ Registered Ports : 1024 - 49151
+ Dynamic/Private Ports : 49152 - 65535

## Port Numbers

### Well-known
+ 22 : SSH
+ 80 : HTTP
+ 443 : HTTPS, UDP
+ 445 : SMB

### Registered
+ 3389 : RDP
