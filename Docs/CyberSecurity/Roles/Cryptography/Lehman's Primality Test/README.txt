# Lehman's Primality Test

### System
+ OS: Windows 10
+ Language: Python 3.10.0

### Application
- Intended Output: 
    + Display all results from the calculations, as well as the number of tries
        ```
        Base: 13
        Tries: 10

        Test [9]: 1
        Test [8]: 1
        Test [7]: 12
        Test [6]: 12
        Test [5]: 1
        Test [4]: 12
        Test [3]: 12
        Test [2]: 12
        Test [1]: 12
        Test [0]: 12

        13 may be Prime.
        ```

## Setup
### Dependencies
+ python3
- Python packages/libraries/modules
    + random: Built-in
