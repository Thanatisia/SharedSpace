## Computer Architectures
+ x86 : 32-bit
    - Also Known As : i386
    - Endian : Big-Endian
+ x64 : 64-bit
    - Also Known As : amd64
    - Endian : Big-Endian
+ x86_64 : 32-bit + 64-bit
    - Endian : Big-Endian
+ ARM : 32-bit
+ ARMv6
+ ARM64 : 64-bit
