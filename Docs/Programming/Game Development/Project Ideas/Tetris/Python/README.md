### Pre-Requisites
- Create Virtual Environment
    ```console
    python -m venv env
    ```

- Chroot into Virtual Environment
    - Linux
        ```console
        . env/bin/activate
        ```
    - Windows
        ```console
        .\env\Scripts\activate
        ```

- Install dependencies
    ```console
    python -m pip install -Ur requirements.txt
    ```
