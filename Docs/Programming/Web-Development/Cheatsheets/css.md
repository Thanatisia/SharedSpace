## Tips n Tricks

### Importing other css file into a css file

+ Using @import
    ```css
    @import url("/path/to/file-name.css");
    @import "file-name.css";
    ```

### Directly importing any or multiple css file(s) inline directly

```html
<head>
    <style>
        @import "file-name.css";
        @import url("/path/to/file-name.css");
    </style>
</head>
```
