# Python - String

## String Formatting
```
In Python, Strings can be formatting and be displayed in various methods, namely

- In-Line,
- .format() function and 
- concatenation
```

- In-Line Formatting
    ```python
    print(f"{variable}")
    ```

- Formatting
    ```python
    print("{}".format(variable))
    ```

- Concatenation
    ```python
    print("Hello" + "World")
    ```
