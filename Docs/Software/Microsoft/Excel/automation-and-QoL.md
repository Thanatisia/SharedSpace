## Keymappings

### General
+ Alt : Access options menu
+ Alt + H : Access Home Tab Operations

### Formatting
+ Alt + H + O : Cell/Grid/Text Formatting
+ Alt + H + O + I : Auto-adjust Width
+ Alt + H + O + A : Auto-adjust Height
