# NixOS Package: nixos-install-tools

## Information
### Background
+ A Collection of NixOS Installation Tools used to Bootstrap install the NixOS base/root filesystem from the command line (like in ArchLinux via pacstrap or Debian via Debootstrap)

### Tools
+ [nixos-enter](nixos-enter.md)
+ [nixos-generate-config](nixos-generate-config.md)
+ [nixos-install](nixos-install.md)
