### Pre-Requisites

- python

### Installing

#### By Package Manager
- If your Package Manager is
	- Debian-based
		```bash
		sudo apt(-get) install xonsh
		```

	- Arch-based
		```bash
		sudo pacman -S xonsh
		```

#### By Python-PyPI

- General
	```
	{python -m} pip install xonsh
	```

- Full with Dependencies
	```
	{python -m} pip install 'xonsh[full]'
	```
